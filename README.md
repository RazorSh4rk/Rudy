# Rudy

## What this _is_

It's a coreXY 3d printer that is

  - cheap
  - reliable
  - small
  - easy to build
  - easy to understand
  - easy to modify
  - open frame

## What this _ain't_

- The fastest printer in the west

- A zero compromise machine

- Something off the shelf
